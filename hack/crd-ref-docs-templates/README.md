# crd-ref-docs templates

Custom Markdown templates for [`elastic/crd-ref-docs`](https://github.com/elastic/crd-ref-docs),
used by `make api-docs`.

Based on the upstream `templates/markdown` set from crd-ref-docs v0.3.0, with the
**Default** column removed (this CRD does not use `kubebuilder:default`).
